Descripción del Proyecto
Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI para el análisis de ventas de una farmacia. El objetivo principal es visualizar indicadores clave de rendimiento (KPIs) y facilitar la toma de decisiones mediante el análisis de datos comerciales.
El panel permite explorar la información de manera dinámica a través de filtros por fecha y visualizaciones que muestran la evolución de las ventas, la composición por formas de pago y la distribución de importes gravados y exentos.

Objetivos
Analizar el desempeño comercial de la farmacia.
Identificar tendencias de ventas en el tiempo.
Evaluar la participación de los distintos medios de pago.
Comparar ventas gravadas y exentas.
Obtener indicadores clave como ticket promedio y cantidad de operaciones.

Indicadores Principales (KPIs)
Total de Ventas: $49.000.000
Total de Operaciones: 2.342
Ticket Promedio: $18.000

Visualizaciones Incluidas
Evolución de ventas por fecha.
Ventas por forma de pago (efectivo, tarjetas, obra social).
Distribución porcentual de medios de pago.
Ventas gravadas vs exentas.
Segmentación temporal (año, trimestre, mes y día).

Herramientas Utilizadas
Power BI Desktop
Modelado de datos
Medidas DAX
Visualizaciones interactivas

Dataset
El conjunto de datos contiene información de operaciones diarias, incluyendo:
Fecha
Número de operaciones
Importes brutos y netos
Ticket promedio
Ventas gravadas y exentas
Formas de pago

Resultados
El dashboard permite visualizar de forma clara el comportamiento de las ventas, destacando que la mayor proporción de ingresos proviene de operaciones exentas y que el ticket promedio se mantiene en valores estables durante el período analizado.# punto_farma
